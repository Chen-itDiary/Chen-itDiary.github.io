1.给你一棵 完全二叉树 的根节点 root ，求出该树的节点个数。

完全二叉树 的定义如下：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第 h 层，则该层包含 1~ 2h 个节点。

2.层序遍历二叉树

给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

3.反转链表

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

4.移动零

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

5.前K个高频元素

给你一个整数数组 nums 和一个整数 k ，请你返回其中出现频率前 k 高的元素。你可以按 任意顺序 返回答案。

这个题目，我利用了大顶堆（降序），以及哈希表来做。

首先哈希表记录出现次数，而优先队列（大顶堆）实现元素次数排序。（这里需要自己实现一个降序比较器，因为优先队列存储数组不支持直接比较；另外队列中数组只存储两个元素，第一个为元素值，第二个为元素出现的次数），最后根据需要的k个元素，依次从队列顶端poll出数组即可

7寻找二叉搜索树的目标节点

某公司组织架构以二叉搜索树形式记录，节点值为处于该职位的员工编号。请返回第 cnt 大的员工编号。

根据题意其实我们知道这是一座二叉搜索树，意味着我们利用中序遍历是可以获取到一串有序的数字的，而题意要求返回第cnt大的元素值。通过中序遍历二叉搜索树，可以获取到递增的数字值，但这对于我们取得第cnt大元素其实并不方便，因为第cnt大元素是从树中最高元素往回数的第几个元素，与我们获取的递增数字值相反……

那我们现在就有了一种思路，利用中序遍历，遍历每一个值并把其放入集合中，最后返回集合的第cnt大元素即可。

同时我们还有另一种思路，既然正常的中序遍历是获取到递增的数字值，那么反过来呢，是不是就可以获取到递减的数字值了？就是这样的，我们利用反中序遍历（（1）右子树，（2）值，（3）左子树）可以获取到从高到低的数字值，我们既然要获取到第cnt大元素，那么直接对进行到步骤（2）的次数进行判断即可，当次数等于cnt时，表明该结点的值就是我们要找的第cnt大元素，return即可.

8.验证二叉搜索树

给你一个二叉树的根节点 root ，判断其是否是一个有效的二叉搜索树。

有效 二叉搜索树定义如下：

  • 节点的左子树只包含小于 当前节点的数。

  • 节点的右子树只包含 大于 当前节点的数。

  • 所有左子树和右子树自身必须也是二叉搜索树。

二叉搜索树（排序树），是一组通过中序遍历树后可以获得的有序的递增的值。

依照这个思路，我们不妨利用中序遍历遍历给予我们的树，并将每个值都存储在List集合中，最后在遍历集合查看是否是递增有序即可判断该树是否是二叉搜索树（排序树）

9.单调递增的数字

当且仅当每个相邻位数上的数字 x 和 y 满足 x <= y 时，我们称这个整数是单调递增的。

给定一个整数 n ，返回 小于或等于 n 的最大数字，且数字呈 单调递增 。

对于这个题目，刚开始看到思考一会后其实就会有这样的思路:

找到最开始下降的那个点—>将该点数字值-1—>其后的所有数字全部变为9—>获取到题意要求的单调递增数字

这看起来是简单的，但实际代码操作起来有些地方需要注意（我是刷题小菜，这题代码写的好冗余！！！哭了）。

注意点比如，在寻找下降的点时，如果存在重复的数，就只需要记住第一个重复的位置，例如332，只需要记录第一个3出现的位置；

还有就是如果既存在重复数，下降点又不在重复数后，例如334452，此时还需要比较最后一个重复数字出现位置和下降点位置的比较（这是我代码的思路，感觉真的好冗余，有些bug是根据测试后有问题改的）。

10.亲密字符串

给你两个字符串 s 和 goal ，只要我们可以通过交换 s 中的两个字母得到与 goal 相等的结果，就返回 true ；否则返回 false

交换字母的定义是：取两个下标 i 和 j （下标从 0 开始）且满足 i != j ，接着交换 s[i] 和 s[j] 处的字符

  • 例如，在 "abcd" 中交换下标 0 和下标 2 的元素可以生成 "cbad"

对于亲密字符串，细读题意：只通过一次交换，即可让两个字符串equals。

所以我们分析满足equals的条件即可:

1.长度相等，只存在两处字符不同

11.反转字符串中的单词

给你一个字符串 s ，请你反转字符串中 单词 的顺序。

单词 是由非空格字符组成的字符串。s 中使用至少一个空格将字符串中的 单词 分隔开。

返回 单词 顺序颠倒且 单词 之间用单个空格连接的结果字符串。

注意：输入字符串 s中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

13.二叉树的最大深度（DFS实现）

给定一个二叉树 root ，返回其最大深度。

二叉树的 最大深度 是指从根节点到最远叶子节点的最长路径上的节点数。

本题利用DFS深度优先搜索来做，即遍历每一个结点，但是只取长路径上的结点\

14.翻转二叉树

给你一棵二叉树的根节点 root ，翻转这棵二叉树，并返回其根节点。

对于这道题，我利用了递归，目的在于实现每个结点的左右子树调换（主要从树的底端结点依次向上进行调换）

15.二叉树的所有路径

给你一个二叉树的根节点 root ，按 任意顺序 ，返回所有从根节点到叶子节点的路径。

叶子节点 是指没有子节点的节点。

由于近期在学习DFS，所以这道题还是运用DFS深度优先搜索来做吧.

题目要求把所有结点路径都记录下来，因此是可以利用DFS遍历所有结点及回溯的特点的

16.二叉树的最小深度

给定一个二叉树，找出其最小深度。

最小深度是从根节点到最近叶子节点的最短路径上的节点数量。

说明：叶子节点是指没有子节点的节点。

这题最小搜查最小路径可以用到BFS的思想，对于二叉树来说也就是层序遍历
